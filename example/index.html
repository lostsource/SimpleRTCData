SimpleRTCData
browser-to-browser messaging simplified

SimpleRTCData is a tiny open source JavaScript library which can be used to establish a standard RTCDataChannel between two peers. It does not handle relaying of messages during connection setup so a separate signalling mechanism is required. Documentation is available on GitHub and an example where you can use copy+paste to simulate the initial signalling message exchange between two peers is available below.
